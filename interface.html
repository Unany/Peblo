Functional Requirements

1 - Start Screen The selection on the start screen uses a simple highlighted selection. 1:1 - Load The load screen is able to move across both memory card slots and be able to pick from multiple saves. 1:2 – Continue Clicking on continue will immediately take you into your last game save by date/time.



2 - In Game Pressing start will bring up the in game menu. 2:1 – Status The status section is not interactive and shows up the avatars health. 2:2 – Equipment The equipment shows what you currently have equipped. This is interactive and clicking on the piece of equipment activates the command section.
